### Docs-site gotchas already solved (don't re-debug)
- pnpm v11 blocks build scripts → `website/pnpm-workspace.yaml` `allowBuilds: {@swc/core, core-js, @parcel/watcher, postman-code-generators}: true`.
- Generated OpenAPI tag pages import `@docusaurus/theme-common` → added it as a **direct dep** (pnpm isolation).
- TypeDoc must NOT use the app's root `tsconfig` (pulls all `.tsx`, needs React deps absent in the docs build) → **scoped `website/tsconfig.typedoc.json`** + `skipErrorChecking`.
- `showLastUpdate*` throws without a `.git` worktree → gated on `fs.existsSync('../.git')`.
- Root `prepare: lefthook install` fails on hosts without git → now `lefthook install || true`.
